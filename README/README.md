# 🐶 Dog Breed Identifier

A Python-based project for identifying dog breeds from DNA sequences and generating a phylogenetic tree.  
This project demonstrates **bioinformatics workflows** such as sequence comparison, breed mapping, and visualisation using Biopython.

---

## 🚀 Features
- 🔍 **Breed Identification**: Finds the closest matching dog breed from input DNA sequences.
- 🧬 **Sequence Comparison**: Uses pairwise alignment to calculate percent identity between sequences.
- 🌳 **Phylogenetic Tree**: Automatically generates a tree visualisation to show relationships between breeds.
- 📂 **Breed Mapping**: Automatically updates or creates a mapping file for known and unknown breeds.
- ✅ Fully **tested** with `pytest`.

---

## 🛠️ Tech Stack
- Python 3.12
- [Biopython](https://biopython.org/)
- `pytest`
- `matplotlib`
